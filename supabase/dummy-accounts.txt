── How to create these accounts ─────────────────────────────────────────
1. Go to https://www.thediscombill.com/login/ (or your local preview) and
   use "Create account" for each row above — full name, email, and the
   shared password. This calls Supabase auth.signUp() and creates a
   `profiles` row automatically (role defaults to 'user').

   If "Confirm email" is enabled in your Supabase project (Authentication
   → Providers → Email), each account also needs its confirmation link
   clicked before it can sign in. For testing, you may want to turn that
   setting OFF temporarily.

2. Promote the 3 expert accounts by running this in Supabase → SQL Editor
   AFTER they've signed up (their profiles row must already exist):

   update public.profiles
   set role = 'expert'
   where id in (
     select id from auth.users where email in (
       'anjali.rao.expert@thediscombill.com',
       'manoj.kumar.expert@thediscombill.com',
       'deepa.nair.expert@thediscombill.com'
     )
   );
